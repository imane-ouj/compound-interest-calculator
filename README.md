💰 Compound Interest Calculator (Python)

📌 Description

This is a simple Python program that calculates compound interest yearly.
The user enters the principal amount, interest rate, and time, and the program calculates the final amount.

---

🧮 Formula Used

A = P(1 + r) ^ t

Where:

- P = Principal (initial amount)
- r = Interest rate (in percentage, e.g. 5 for 5%)
- t = Time (in years)
- A = Final amount

---

🚀 Features

- Calculates compound interest (yearly)
- Converts percentage to decimal (rate / 100)
- Simple and beginner-friendly
- Uses basic Python concepts

---

🛠️ How to Use

1. Run the program
2. Enter:
   - Principal amount
   - Interest rate (e.g. 5 for 5%)
   - Time (years)
3. The program will display the final amount

---

💻 Example

Input:
Principal: 1000
Rate: 5
Time: 2

Output:
Final amount: 1102.5
